# Scripts

Place new standardized Croce real-data validation runners and helper utilities here.

The target audience of this directory is the motor-imagery validation workflow defined in [docs/CROCE2017_REAL_DATA_VALIDATION_PLAN.md](../../docs/CROCE2017_REAL_DATA_VALIDATION_PLAN.md).

Existing exploratory scripts under `experiments/scripts/` can be used as references, but new maintained entrypoints should live here.
