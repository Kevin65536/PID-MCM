# Dealing with modality heterogeneity

## Introduction

Modality heterogeneity is a common issue in multi-modal learning. We are investigating methods to address this issue in multi-modal physiological signal area, where different modalities (e.g., EEG and fNIRS) have different sampling rates, signal characteristics, electrode/channel placement, and noise profiles.

## Methods

Current works involved with modality heterogeneity can separate into three main categories:

1. Unified representation in wave form level:
   - PhysioWave
   - Core idea: Use a learnable wavelet transform to replace fixed filters, map different modalities into a unified combination of sub-bands
2. Unified representation in token level:
    - PhysioOmni
       - Core idea: Use a learnable tokenization module to map different modalities into a unified token space, allowing for cross-modal interactions and fusion
    - NeuroRVQ
       - Core idea: A scalable Large Brainwave Model centered on a codebook-based tokenizer for EEG. It features multi-scale feature extraction for the full frequency spectrum, hierarchical residual vector quantization (RVQ) codebooks for high-resolution encoding, and a phase- and amplitude-aware loss function for high-fidelity signal reconstruction.
3. Unified latent space:
    - Brain-OF
       - Core idea: An omnifunctional foundation model jointly pretrained on fMRI, EEG, and MEG. Uses an **Any-Resolution Neural Signal Sampler** to project signals of different spatiotemporal resolutions into a shared semantic space. The backbone integrates DINT attention with a **Sparse Mixture of Experts** (MoE), where shared experts handle modality-invariant features and routed experts handle modality-specific semantics. Pretrained with a Masked Temporal-Frequency Modeling objective.
    - NOBEL
       - Core idea: A neuro-omni-modal brain-encoding large language model (LLM) aligning EEG, MEG, and fMRI into a shared LLM semantic embedding space. Uses a unified encoder for EEG/MEG and a novel dual-path strategy for fMRI, unlocking synergistic fusion of electromagnetic (high-frequency) and metabolic (low-frequency) signals.

## Thoughts

> Modality heterogeneity won't disappear when different modalities are simply mapped into a unified representation space. They can still have different distributions, noise characteristics, vary in semantic information. How to further align them, by training strategies or other ways?
> Different modalities has different time resolution. How to align time resolution difference to semantic information difference?
> Different modalities are like different languages. Physiological signals with different kinds of noises are articles with typos or defacements. Can we explicitly mark the noise or uncertainty of different modalities, and let the model learn to handle them?
